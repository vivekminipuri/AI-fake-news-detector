# AI Fake News & Misinformation Detector

An AI-powered platform to verify news articles and online content, providing a credibility score and explanation using a hybrid AI/NLP approach.

## Structure

- **frontend/**: React + TypeScript application.
- **backend/**: FastAPI REST API.
- **ai-engine/**: NLP models and heuristics pipelines.
- **scripts/**: Utility scripts for setup and maintenance.
